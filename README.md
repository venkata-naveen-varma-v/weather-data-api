# weather-data-api

- This is for the COMP8117 Final Project
- A nodejs, express API that interacts with OpenWeather-API and get the weather details based on the place/city name.
- The location name must be passed as part of the url

Example: https://weatherdata-api-project.herokuapp.com?location=canada here 'location' is the parameter key and 'canada' is the value of that key


Postman API documentation --> https://documenter.getpostman.com/view/15945431/2s83zmKMha
